🏆 골드&실버 실시간 시세 대시보드 (Pro Analyst)
"복잡한 API 키 설정 없이, HTML 파일 하나로 끝내는 나만의 금 시세 분석기"
이 프로젝트는 국제 금/은 시세(XAU/XAG)와 원달러 환율(USD/KRW)을 실시간으로 연동하여, **한국형 금 시세(1돈, 1g, 매매가)**를 자동으로 계산해주는 웹 대시보드입니다.
(스크린샷 이미지를 여기에 넣어주세요)
✨ 주요 기능 (Key Features)
📊 실시간 차트 & 환율 자동 연동
TradingView 차트: 월봉 캔들 차트로 장기 추세를 한눈에 파악.
환율 자동 수신: 무료 API를 통해 접속 시점의 실시간 환율을 자동으로 가져옵니다. (수동 수정 가능)
🧮 스마트 시세 계산기
국제 시세($/oz)를 입력하면 즉시 **한국 시세(₩)**로 변환.
1돈(3.75g), 1그램(1g), 1kg(Bar) 단위 자동 계산.
실거래가 추정: 부가세(VAT) 포함 '살 때' 가격과, 95% 수준의 '팔 때' 가격 자동 산출.
📒 나만의 시세 장부 (My Ledger)
로컬 저장소(Local Storage) 활용: 별도 DB 없이 브라우저에 데이터를 저장.
일별 '살 때/팔 때' 시세를 직접 기록하고 관리 가능.
새로고침 해도 데이터가 날아가지 않습니다.
📱 반응형 웹 디자인
Tailwind CSS를 사용하여 PC, 태블릿, 모바일 어디서든 완벽한 레이아웃 제공.
🚀 사용 방법 (How to use)
이 저장소(Repository)를 Clone 하거나 ZIP으로 다운로드 받으세요.
index.html 파일을 더블 클릭하여 실행하면 끝! (서버 불필요)
GitHub Pages에 올리면 나만의 웹사이트로 무료 배포 가능합니다.
